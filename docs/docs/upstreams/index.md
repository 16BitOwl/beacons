# Upstreams

Upstreams define where DNS records are pushed. Each is a named instance,
referenced by name from source records. You can define multiple of the same
type.

- [Cloudflare](./cloudflare.md)
- [Pi-hole](./pihole.md) — v6+
- [Technitium](./technitium.md)
